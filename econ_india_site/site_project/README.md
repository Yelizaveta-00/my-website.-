Короткі кроки для розміщення на GitHub Pages:
1) Створіть репозиторій на GitHub (наприклад, "econ-india-site") та завантажте в корінь файли з цього архіву.
2) У налаштуваннях репозиторію (Settings) увімкніть Pages → Source: main branch / root (або виберіть gh-pages branch) — збережіть.
3) Після кількох хвилин сайт буде доступний за адресою https://<your-username>.github.io/<repo-name>/
